This is the first post on the new ry4nolson.com. The site is a personal portfolio and blog: a place to list projects (like [The Annex](https://theannex.app) and [Cutover](https://cutoverapp.dev)), share a bit about what I do, and occasionally write.

**What you’ll find here**

- **Home** — Short intro and links to projects and blog
- **About** — Bio, ry4nolson LLC, what I build, and contact
- **Projects** — List of things I’ve built or am building
- **Blog** — Posts like this one, written in Markdown and built at deploy time

The stack is React 19, React Router v7 with SSR, Tailwind CSS, Vite, and Netlify. Blog posts live as Markdown files in the repo with frontmatter; no CMS. If you’re reading this, the pipeline worked.

Thanks for stopping by.
